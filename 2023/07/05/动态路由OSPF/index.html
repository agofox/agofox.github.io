建立邻居的过程

7种状态：

  1.  down:  没有宣告 接口down

  2.  init:  
     
 互发hello 
 router-id      标识设备在ospf中的名称
                选举  1.手动     router ospf 1 
                                 router-id 2.2.2.2
                      2.逻辑接口IP地址最大的
                      3.物理接口IP地址最大的

  3.  2-way:   
       选举DR： 1.DR优先级，越大越优 范围0-255 默认为1，为0时不选举
                2.router-id 越大越优
                3.  int f0/0   ip ospf priority 0
              
       DR选举很慢:     10-40s
                       int f0/0
                       ip ospf network point-to-point

  4.  exstart

  5.  exchange

  6.  loading

  7.  full

重置进程 clear ip ospf processy
